Тема: параллельное вычисление среднего значения массива с использованием OpenMP.

Суть работы:

Создаётся большой массив чисел.

Среднее значение считается параллельно с помощью OpenMP при разном числе потоков.

Каждый поток суммирует свою часть массива, после чего результаты объединяются.

Ключевой момент:

Показан эффект false sharing: когда разные потоки записывают данные в одну cache line, производительность падает.

Для устранения false sharing используется padding (выравнивание данных до размера cache line).

Результат:

При увеличении числа потоков время выполнения уменьшается.

Версия с padding показывает более стабильное и быстрое выполнение.

Измеряется и анализируется ускорение относительно одного потока.
